提交百度站长

对于 hexo 博客来说，如果你是部署在 GitHub Pages，那么你是无法被百度收录的，因为 GitHub 禁止了百度爬虫，最常见的解决办法是双线部署到 Coding Pages 和 GitHub Pages。可以参考《Hexo 双线部署到 Coding Pages 和 GitHub Pages 并实现全站 HPPTS》

TIP

如果选择 Coding Pages部署的话，我们可以选择境内和境外服务器，但是境外只能选择香港的服务器，而香港服务器CDN加速很慢，甚至不如GitHub Pages的访问速度，唯一的一点就是绑定自己域名的话不需要备案；而选择境内服务器的话，绑定自己域名需要备案。这个就需要你自己做出抉择了。
